# Visual Studio Code (VS Code)

Bu klasör, **VS Code** editörünü yazılım geliştirme sürecinde verimli kullanmak için gereken temel ayar ve uzantı önerilerini içerir.

## Önerilen Uzantılar
- GitLens
- ESLint
- Prettier
- Python
- Docker

## Örnek Ayar (`.vscode/settings.json`)
```json
{
  "editor.formatOnSave": true,
  "files.trimTrailingWhitespace": true,
  "editor.tabSize": 2,
  "javascript.validate.enable": false
}
```

## Kısa Yollar
- Komut paleti: `Ctrl+Shift+P`
- Dosya arama: `Ctrl+P`
- Satır arama: `Ctrl+F`
