# IntelliJ IDEA

Bu klasör, özellikle **Java / Kotlin** projelerinde kullanılan güçlü IDE **IntelliJ IDEA** hakkında kısa notlar içerir.

## Özellikler
- Akıllı kod tamamlama
- Refactoring araçları
- Entegre Maven/Gradle desteği
- Debugger ve test entegrasyonu

## Yaygın Kısayollar (Varsayılan)
- `Ctrl+Shift+A` → Action arama
- `Shift` x2 → Her şeyi ara (Search Everywhere)
- `Ctrl+Shift+F10` → Çalıştır

## Örnek Kullanım
- Maven projesini aç: `pom.xml` dosyasını IDEA ile aç
- Testleri çalıştır: `src/test` klasöründeki test dosyasına sağ tık → **Run tests**
