# Jira

Bu klasör, yazılım projelerinde görev ve süreç yönetimi için kullanılan **Jira** aracı hakkında kısa notlar içerir.

## Temel Kavramlar
- Project
- Issue (Task, Bug, Story)
- Sprint
- Board (Scrum / Kanban)

## Yaygın Kullanım
- Backlog oluşturma
- Görevlere atama (assignee)
- Durumlar: To Do → In Progress → Done

Öğrenciler için genelde Trello/Notion yeterlidir, fakat kurumsal ortamlarda Jira standarttır.
