# Gerçek Prompt Örnekleri

Buraya gerçek hayatta işine yarayan **iyi çıkan prompt + cevap** örneklerini koyabilirsin.

Önerilen kullanım:
- Her dosya bir konu: `ornek-01-test-uretme.md`, `ornek-02-debug-yardimi.md` gibi.
- İçine hem kullandığın prompt'u hem de aldığın cevabın önemli kısımlarını koy.
- Yanına kendi yorumunu yaz ("şu kısım çok işime yaradı", "şu kısmı bir dahaki sefere daha net sormalıyım" gibi).

Bu klasör tamamen senin kişisel arşivin.
