# Prompt Engineering Lablar

Bu klasör, kendin için hazırladığın uygulamalı çalışmalar (lab) için.

Örnek lab fikirleri:
- **Lab 01:** Kötü/iyi prompt karşılaştırması (kod açıklatma)
- **Lab 02:** Unit test üretimi (pytest veya Jest)
- **Lab 03:** Mevcut kodu refactor ettirme
- **Lab 04:** Hata analizi ve debug yardımı
- **Lab 05:** API için dokümantasyon yazdırma

Her lab için ayrı bir dosya açabilirsin (örneğin `lab-01-kod-aciklama.md`).
