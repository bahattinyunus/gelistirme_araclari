# Prompt Engineering

Bu klasör, üretken yapay zekayı (GenAI) verimli kullanmak için **prompt engineering** üzerine odaklanır.

## Amaç
- Daha net ve kontrollü çıktılar almayı öğrenmek
- Yazılım geliştirme senaryolarında (kod, test, doküman, analiz) güçlü prompt'lar tasarlamak
- İyi/kötü prompt örneklerini karşılaştırmak

## İçerik Fikri
- Temel ilkeler (bağlam, rol, format, adım adım düşünme, iteratif iyileştirme)
- Uygulamalı egzersizler (mini lab'lar)
- Ödev/çalışma soruları

İlerleyen adımlarda bu klasörde ayrıntılı **lab** ve **örnek prompt setleri** oluşturacağız.
