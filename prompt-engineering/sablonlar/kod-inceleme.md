# Kod İnceleme Şablonu

"Kıdemli bir [dil/framework] geliştiricisi gibi davran. Aşağıdaki kodu incele:

```[dil]
// kod buraya
```

Şunları istiyorum:
1. Genel bir özet ver (kod ne yapıyor?).
2. Potansiyel bug veya edge-case var mı, belirt.
3. Okunabilirlik / performans / güvenlik açısından öneriler yaz.
4. İyileştirilmiş bir sürüm öneriyorsan, sadece önerdiğin kısmın kodunu ver.
"
